LogHawk

Effortless log triage, real-time detection, and IOC enrichment.

View on GitHub

Key Features

  • Parse and search logs rapidly
  • Highlight suspicious activity
  • Auto-enrich IPs/domains with threat intelligence
  • Designed for SOC analysts and threat hunters

Get Started

Clone the repository and run LogHawk with your logs:

git clone https://github.com/Imane-Ismail/LogHawk.git
cd LogHawk
python loghawk.py --logfile logs/syslog.log
