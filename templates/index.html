🦅 LogHawkX Analyzer

About LogHawkX:
LogHawkX is a lightweight threat detection tool that scans uploaded log files for indicators of suspicious and malicious activity — such as brute force, PowerShell abuse, lateral movement, encoded payload, Command & Control or potential web-based attacks.

Supported File Types:
• .log
• .txt
• .json
• .csv

File Size Limit:
Files up to 5 MB are supported.

How to Use:
Upload a supported log file or simply paste the logs below. LogHawkX will analyze it and display any detected security events or anomalies.
